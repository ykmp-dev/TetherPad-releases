# TetherPad Releases

AltStore source for [TetherPad](https://github.com/ykmp-dev/TetherPad), an FTP tether-capture app for iPad.

## Install

1. Install [AltStore](https://altstore.io) on your iPad (AltServer on PC required).
2. In AltStore, tap `Browse` → `+` → paste:

   ```
   https://ykmp-dev.github.io/TetherPad-releases/source.json
   ```

3. Install TetherPad from the source.

## Requirements

- iPad running iOS 16 or later
- Apple ID (free tier works — 7-day refresh via AltServer)
- Wi-Fi network shared with your camera (for FTP mode)

## What's here

- `source.json` — AltStore source manifest
- `TetherPad.ipa` — latest build
- `index.html` — landing page
